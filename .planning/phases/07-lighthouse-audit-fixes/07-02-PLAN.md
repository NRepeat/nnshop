<objective>
Add `aria-label` to icon-only social links and fix footer copyright text contrast ratio.

Purpose: The Telegram and Viber links in the announcement bar contain only SVG icons with no visible text. Screen readers and Lighthouse report these links as having no accessible name (A11Y-01). The footer copyright line uses `text-white/40` on a `#1a1a1a` background, producing a 3.83:1 contrast ratio which fails WCAG 2.1 AA (requires ≥ 4.5:1). Changing to `text-white/50` produces 5.24:1 (A11Y-02).

Output: Updated announcement-bar.tsx with `aria-label` on both social link anchors, and updated Footer.tsx with `text-white/50` on the copyright div.
</objective>
