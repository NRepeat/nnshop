  <action>
Read the current file first.

The trigger button (line ~59) currently has:
```typescript
className="h-full border-b-2 border-foreground bg-foreground hover:border-b-2 hover:border-b-foreground transition-colors"
```

The `border-b-2`, `border-foreground`, `hover:border-b-2`, `hover:border-b-foreground` classes are redundant — they create a border the same color as the button background, which is invisible, and hover makes no change. `transition-colors` with no actual color change is also a no-op here.

Replace the className with just the height class:
```typescript
className="h-full"
```

The button `variant="default"` (from shadcn/ui) already provides background, text color, hover state, and focus ring styling. The `h-full` keeps the button flush with the header bar height.

Do not change any other part of the component.
