<objective>
Remove all PII-bearing console.log calls from the four highest-risk files: auth.ts, on-link-account.ts, anonymous-cart-buyer-identity-update.ts, and create.ts.

Purpose: These files log user IDs, email addresses, and partial cart tokens to server logs during auth and cart merge flows. In production this constitutes a PII leak visible in platform logs (Vercel, server stdout). Removing these logs closes the leak with zero functional change.
Output: Four edited source files with all debug console.log calls removed; console.error in catch blocks preserved for error observability.
</objective>
