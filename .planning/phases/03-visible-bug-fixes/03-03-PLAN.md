    **Step 1 — Create `src/shared/store/use-cart-ui-store.ts`:**

    Follow the existing Zustand store pattern in `src/shared/store/` (see `use-cart-note-store.ts` or `use-fav-store.ts` for reference — they use `create` from `zustand`). Do NOT use `persist` middleware — cart sidebar open state is transient (no localStorage persistence needed).

    ```typescript
    'use client';

    import { create } from 'zustand';

    interface CartUIState {
      isOpen: boolean;
      openCart: () => void;
      closeCart: () => void;
    }

    export const useCartUIStore = create<CartUIState>((set) => ({
      isOpen: false,
      openCart: () => set({ isOpen: true }),
      closeCart: () => set({ isOpen: false }),
    }));
    ```

    **Step 2 — Create `src/features/header/cart/ui/CartSheetController.tsx`:**

    This is a thin client component that reads `isOpen` and `closeCart` from `useCartUIStore` and passes them to the `<CartSheet>` server component via props. The `CartSheet` component passes `{...props}` through to the underlying shadcn `<Sheet>` which is a Radix Dialog Root that supports `open` and `onOpenChange` props for controlled mode.

    ```typescript
    'use client';

    import { useCartUIStore } from '@shared/store/use-cart-ui-store';
    import CartSheet from './Sheet';

    interface CartSheetControllerProps {
      locale: string;
    }

    export const CartSheetController = ({ locale }: CartSheetControllerProps) => {
      const { isOpen, closeCart } = useCartUIStore();
      return (
        <CartSheet
          locale={locale}
          open={isOpen}
          onOpenChange={(open) => {
            if (!open) closeCart();
          }}
        />
      );
    };
    ```

    **Step 3 — Update `src/features/header/ui/HeaderOptions.tsx`:**

    - Remove: `import CartSheet from '../cart/ui/Sheet';`

    - Replace: `<CartSheet locale={locale} />` → `<CartSheetController locale={locale} />`

    The `<Suspense>` wrapper around it stays unchanged.

    **Important:** `CartSheet` in `Sheet.tsx` is an `async` Server Component. It already passes `{...props}` spread to `<Sheet>` (the shadcn wrapper around Radix Dialog Root). Radix Dialog Root's `open` and `onOpenChange` props enable controlled mode without any changes to `Sheet.tsx` itself.

    **Step 3 — Wire size buttons:**

    ```typescript
    {sizeOptions && (
      <div className="mt-4">
        <div className="flex gap-2 mt-1">
          {sizeOptions.map((size) => {
            // Find the variant matching this size
            const matchingVariant = product.variants.edges.find((edge) =>
              edge.node.selectedOptions.some(
                (option) =>
                  option.name.toLowerCase() === 'розмір' &&
                  option.value.toLowerCase() === size.toLowerCase(),
              ),
            )?.node ?? null;

            const isSelected =
              selectedVariant !== null &&
              selectedVariant.selectedOptions.some(
                (option) =>
                  option.name.toLowerCase() === 'розмір' &&
                  option.value.toLowerCase() === size.toLowerCase(),
              );

            const isOutOfStock = matchingVariant
              ? !matchingVariant.availableForSale
              : false;

            return (
              <Button
                key={size}
                variant={isSelected ? 'default' : 'outline'}
                disabled={isOutOfStock}
                className={isOutOfStock ? 'opacity-50 line-through cursor-not-allowed' : ''}
                onClick={() => {
                  if (!isOutOfStock && matchingVariant) {
                    setSelectedVariant(matchingVariant);
                  }
                }}
              >
                {size}
              </Button>
            );
          })}
        </div>
      </div>
    )}
    ```
