## User Setup Required

To enable the Viber link before a Sanity `viberPhone` value is published, add to `.env.local`:

```
VIBER_PHONE_NUMBER=380991234567
```

Use raw digits only (no `+`, no spaces). Once `viberPhone` is set in Sanity Studio (`Site Settings > Info Bar`), the Sanity value takes precedence over the env var.
